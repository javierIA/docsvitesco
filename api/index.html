Instalación del Proyecto en Minimal API

La Minimal API es una característica reciente de ASP.NET Core que permite crear APIs web con una sintaxis más concisa y minimalista. Si deseas instalar y ejecutar el proyecto usando Minimal API, sigue los pasos a continuación:

Requisitos Previos

  1. .NET SDK: Debes tener instalado el SDK de .NET 6 o posterior.

    • Puedes verificar la versión instalada con el comando: 
      dotnet --version
    • Si no lo tienes instalado, descárgalo desde el sitio oficial de .NET.

Paso 1: Clonar el Repositorio

  1. Abre tu terminal o línea de comandos.
  2. Navega al directorio donde deseas clonar el proyecto.
  3. Utiliza el siguiente comando para clonar el repositorio: 
    git clone URL_DEL_REPOSITORIO

Paso 2: Navegar a la Carpeta del Proyecto

  1. Una vez clonado el repositorio, navega a la carpeta del proyecto con el comando: 
    cd NOMBRE_DE_LA_CARPETA_DEL_PROYECTO

Paso 3: Instalar las Dependencias

  1. Si el proyecto utiliza un archivo NuGet.Config o tiene dependencias adicionales, puedes restaurarlas usando: 
    dotnet restore

Paso 4: Ejecutar la Aplicación

  1. Con todo listo, ejecuta la aplicación con el comando:

    dotnet run

  2. Una vez que la aplicación esté ejecutándose, deberías ver un mensaje indicando que el servidor está escuchando, algo así como:

    Now listening on: http://localhost:5000

  3. Abre un navegador y ve a http://localhost:5000 para acceder a la aplicación.

Paso 5 (Opcional): Configuración Adicional

Dependiendo del proyecto, es posible que debas realizar pasos adicionales, como configurar variables de entorno, bases de datos o servicios externos. Asegúrate de revisar cualquier documentación adicional o archivos README incluidos en el proyecto para obtener instrucciones específicas.

Consideraciones Finales

  • La Minimal API es ideal para proyectos que buscan una implementación rápida y un código más limpio. Sin embargo, si tu proyecto crece considerablemente, podrías considerar migrar a una estructura más completa de MVC o Razor Pages en el futuro.
  • Asegúrate de mantener tu proyecto actualizado con las últimas versiones de las dependencias y el SDK de .NET para aprovechar las mejoras de rendimiento y seguridad.

Espero que esta guía te haya sido útil. ¡Buena suerte con tu proyecto en Minimal API!
