Introduccion

¿Que es flutter?

Flutter es un SDK de código abierto creado por Google para desarrollar aplicaciones para Android e iOS, así como aplicaciones para Google Fuchsia, Linux, Mac, Windows, web y aplicaciones de escritorio desde una única base de código.

¿Que es dart?

Dart es un lenguaje de programación de código abierto, desarrollado por Google. Fue revelado en la conferencia goto; en Aarhus, Dinamarca el 10 de octubre de 2011. El objetivo de Dart no es reemplazar JavaScript como el principal lenguaje de programación web en los navegadores web, sino ofrecer una alternativa más moderna.

Tutorial: Instalación de Flutter en Mac para Desarrollo iOS

Este tutorial te guiará a través de los pasos necesarios para instalar Flutter en una Mac y configurar tu entorno para el desarrollo de aplicaciones iOS.

Requisitos Previos

  • Una Mac con macOS (se recomienda la última versión).
  • Al menos 2.8 GB de espacio libre en disco.

Paso 1: Instala Xcode

Xcode es el entorno de desarrollo integrado (IDE) oficial para el desarrollo en iOS.

  1. Ve al App Store en tu Mac.
  2. Busca "Xcode".
  3. Haz clic en "Obtener" y luego en "Instalar".
  4. Una vez instalado, abre Xcode y sigue las instrucciones para completar la instalación de los componentes adicionales que se requieran.

Paso 2: Instala Flutter SDK

2.1. Descarga el SDK

  1. Ve al sitio oficial de Flutter.
  2. Haz clic en el botón "Get started".
  3. Selecciona "macOS" en la lista de sistemas operativos.
  4. Descarga el archivo flutter_macos_latest-stable.zip.

2.2. Extrae el archivo

  1. Navega a la ubicación donde descargaste el archivo zip.
  2. Extrae el archivo para obtener una carpeta llamada flutter.

2.3. Actualiza tu PATH

Para poder ejecutar el comando flutter desde cualquier terminal, necesitas agregar flutter/bin a tu PATH.

  1. Abre tu terminal.
  2. Edita tu archivo .zshrc (o .bashrc si usas bash) con tu editor favorito, por ejemplo: nano ~/.zshrc.
  3. Agrega al final del archivo: export PATH="$PATH:ruta_a_tu_carpeta_flutter/flutter/bin".
  4. Guarda el archivo y cierra el editor.
  5. Ejecuta source ~/.zshrc para actualizar tu terminal.

Paso 3: Verifica tu instalación

  1. En la terminal, ejecuta el comando flutter doctor.
  2. Flutter doctor revisará cualquier dependencia que pueda faltar y te dará instrucciones sobre cómo solucionar cualquier problema que encuentre.

Paso 4: Configura un simulador de iOS

Con Xcode instalado, puedes usar su simulador para probar tus aplicaciones Flutter.

  1. Abre Xcode.
  2. Ve a Preferences > Components.
  3. Selecciona el simulador que deseas (por ejemplo, iPhone 12) y descárgalo.
  4. Una vez descargado, puedes abrir el simulador seleccionando Open Developer Tool > Simulator.

Paso 5: Abir un proyecto Flutter

  1. Abre tu explorador de archivos.
  2. Navega a la carpeta donde esta proyecto VitescoPushClean (por ejemplo, cd ~/Documents/VitescoPushClean).
  3. Ejecuta flutter run para compilar y ejecutar el proyecto en el simulador.

Paso 6: Instalar dependencias

  1. Abre tu explorador de archivos.
  2. Navega a la carpeta donde esta proyecto VitescoPushClean (por ejemplo, cd ~/Documents/VitescoPushClean).
  3. Ejecuta flutter pub get para instalar las dependencias del proyecto.

Paso 7: Compilar proyecto

  1. Abre tu explorador de archivos.
  2. Navega a la carpeta donde esta proyecto VitescoPushClean (por ejemplo, cd ~/Documents/VitescoPushClean).
  3. Ejecuta flutter build ios para compilar el proyecto.

Paso 8: Ejecutar proyecto en dispositivo

  1. Abre tu explorador de archivos.
  2. Navega a la carpeta donde esta proyecto VitescoPushClean (por ejemplo, cd ~/Documents/VitescoPushClean).
  3. Conecta tu dispositivo iOS a tu Mac.
  4. Ejecuta flutter run para compilar y ejecutar el proyecto en el dispositivo. 4.1. Si no funciona, ejecuta flutter doctor para ver si tienes todo lo necesario para ejecutar el proyecto en un dispositivo iOS. 4.2. Si dice el error de firmado favor de ir a la siguiente sección para solucionarlo.
