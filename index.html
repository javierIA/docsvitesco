API Push Notification - Documentación

Bienvenido a la documentación oficial de la API Push Notification.

Introducción

Esta documentación proporciona información detallada sobre cómo utilizar la API Push Notification. Aprenderás sobre cómo autenticarte, hacer solicitudes y manejar respuestas.

Autenticación

Token de Autenticación

La API utiliza autenticación basada en tokens. Para autenticarte:

  1. Registra una cuenta con nosotros.
  2. Accede a tu panel de control.
  3. Genera un token en la sección de autenticación.
  4. Incluye el token en tus solicitudes a la API en el encabezado Authorization.

Ejemplo de uso:

GET /endpoint
Headers:
Authorization: Bearer YOUR_TOKEN_HERE

End Points

Usuarios

  • GET /users: Obtiene una lista de todos los usuarios.
  • POST /users: Crea un nuevo usuario.
  • PATCH /users/:id: Actualiza un usuario específico.
  • DELETE /users/:id: Elimina un usuario específico.

Dispositivos

  • GET /devices: Obtiene una lista de todos los dispositivos.
  • POST /devices: Crea un nuevo dispositivo.
  • ... (y así sucesivamente para cada end point)

Para obtener detalles sobre cómo usar cada punto final, parámetros, cuerpos de solicitud y respuestas, consulte la documentación detallada.

Ejemplos de Uso

Crear un Usuario

Para crear un usuario, realiza una solicitud POST a /users con el siguiente cuerpo:

{
    "name": "Juan Pérez",
    "email": "juan@example.com",
    "password": "securepassword123"
}

Enviar una Notificación

Para enviar una notificación, realiza una solicitud POST a /notifications con el siguiente cuerpo:

{
    "userId": "123456",
    "message": "Esta es tu notificación"
}

Errores Comunes

  • Error 401 Unauthorized: Asegúrate de que tu token de autenticación es válido y está incluido en el encabezado Authorization.
  • Error 404 Not Found: El recurso que intentas acceder no existe. Verifica la URL y prueba nuevamente.
  • ... (otros errores y soluciones)

FAQ

¿Puedo generar más de un token?

Sí, puedes generar múltiples tokens desde tu panel de control.

¿Cuánto tiempo es válido un token?

Los tokens son válidos durante 30 días a partir de la fecha de creación.
